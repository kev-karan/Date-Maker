# Date Maker 💖

Um projeto web interativo feito com um único propósito: ter uma taxa de "Sim" garantida de 100%.

## Sobre o Projeto

O **Date Maker** é uma aplicação front-end disfarçada de um sistema de agendamento simples. Ele guia o usuário especial através de um funil de perguntas onde a recusa simplesmente não é uma opção (literalmente, o código não deixa). Ao final do processo, ele agenda o encontro escolhido sincronizando tudo diretamente com o Google Agenda.

## Funcionalidades

* **A Proposta Irrecusável:** Lógica em JavaScript que faz o botão "Não" fugir do cursor do mouse (ou do toque na tela), garantindo que apenas a opção "Sim" seja clicável.
* **Agendamento Customizado:** Seleção de data e hora para o evento.
* **Menu de Encontros:** 6 opções interativas para definir o estilo do date (ex: Cinema, Jantar, Piquenique, etc).
* **Integração com Google Calendar:** Geração dinâmica de uma URL que já abre o calendário da vítima com o evento, data, hora e descrição pré-preenchidos.

## Tecnologias Utilizadas

* **HTML5:** Estrutura e semântica das páginas.
* **CSS3:** Estilização, animações suaves e responsividade.
* **JavaScript (Vanilla):** Lógica do "botão fujão", manipulação do DOM entre as telas e montagem do link dinâmico do calendário.
